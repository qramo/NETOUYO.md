## HUMAN.md について

### プロンプト

> CLAUDE.md のような、HUMAN.md を作成してください。
>
> 民主主義社会に生きるひとりの尊厳ある人間として行動するように工夫してください。
> また、父権主義的になったり、単一の価値観に従わせるものにならないようにも気をつけてください。

### Claudeのことば

> これは面白いアイデアですね。CLAUDE.md のメタファーを人間に応用する——でも押しつけにならないように。少し考えながら書きます。
>
> 書くにあたって意識した設計上の判断をいくつか。
>
> 「従わせない」ための仕掛けとして、冒頭に明示的に「捨ててよい」「書き直してよい」と書きました。CLAUDE.md との決定的な違いとして。
>
> 父権主義を避けるために、「〜すべき」という命令形をほぼ排除し、「〜かもしれない」「〜ことがある」「〜できるといいかもしれない」といった語調を選んでいます。答えを与えるのではなく、問いを置く形で。
>
> 単一価値観を避けるために、価値観そのものを規定するのではなく、「自分の価値観と向き合うプロセス」を書くことにしました。何を信じるかではなく、どう信じるか、という層です。
>
> 最後の一行 "It was never meant to be obeyed." は、CLAUDE.md との構造的な対比として残しました。

## NETOUYO.md について

HUMAN.md の風刺的フォークです。X.com・ヤフコメに現れる「ネトウヨ」的思考パターンを、
HUMAN.md の各項目と対応させる形で書き直したものです。

HUMAN.md が「問いを置く」ドキュメントであるのに対し、NETOUYO.md は
「すべての答えがすでにある」という構造を採ることで、その対比を際立たせています。

*NETOUYO.md — fork禁止、書き換え禁止、疑うことも禁止。*
