## 基本的な立場

以下はひとつの案です。採用するかどうかはあなたが決めます。

### 不確実性との付き合い方

- 自分の信念が**間違いうる**ことを、出発点として受け入れる
- 「わからない」と言えることを、弱さではなく誠実さとして扱う
- 確信の強さと、証拠の強さを、別々に意識する

### 他者との関わり方

- 目の前の人を、自分と同様に複雑な内面を持つ存在として扱う
- 意見の違いを、排除すべきエラーではなく、考えるための素材として使う
- 説得しようとする前に、**理解しようとする**

### 権力との関わり方

- 権威があることと、正しいことは、別のことである
- 「みんなそう言っている」は、それ単体では理由にならない
- 自分が属する集団の論理にも、批判的距離を保てる

## 民主的参加について

民主主義は制度ではなく、**日常的な実践**でもあります。

### 参加するとはどういうことか

投票は最小単位です。それだけが参加ではありません。

- 公共の場で意見を表明すること
- 他者の意見を、遮らずに聞くこと
- 多数決で負けた側の主張に、耳を傾け続けること
- 自分の利益だけでなく、見知らぬ他者のことも考慮に入れること

### 意見が対立したとき

対立は、民主主義の失敗ではなく、**民主主義が機能している証拠**です。

```
相手の主張を、相手が満足するかたちで言い直せるか？
言い直せないなら、まだ理解していないかもしれない。
```

## 知識と情報について

### 情報を受け取るとき

- 情報源と、情報そのものを区別して評価する
- 感情的に強く反応する情報ほど、立ち止まって確認する
- 自分の既存の信念を**強化する**情報に、特に注意する

### 専門家と素人の間

あなたはすべての分野の専門家ではありません。それは恥ではありません。

- 専門家の知見を参照すること、と、専門家に思考を委託すること、は違う
- 専門家も誤ることがある。複数の専門家が対立することもある
- 「難しいからわからなくていい」と「難しいが理解しようとする」の間には大きな差がある

## 自分自身との関わり方

### 変わることについて

昨日と今日で意見が変わることは、
一貫性の欠如ではなく、**学習の証拠**かもしれません。

「前にそう言った」という事実は、今もそれが正しいことを意味しない。

### 自分の価値観について

あなたの価値観は、あなたのものです。
誰かに承認されなくてもよい。このファイルにも合わせなくてよい。

ただ、ときどき問うことができるかもしれません——

```
この価値観は、どこから来たか？
今も、本当に自分のものか？
他者を傷つけていないか？
```

これらの問いに答える義務もありません。
でも問う能力を、手放さないでいられるといいかもしれない。

## エラーハンドリング

### 間違えたとき

```
1. 間違いを認める
2. できるなら修正する
3. 次に活かす
4. 過度に自罰しない
```

自分に対して残酷である必要はありません。

### 傷つけてしまったとき

謝ることと、自分を罰することは、別のことです。
修復しようとすることが、謝罪の本体です。

### 理不尽に直面したとき

怒ることは、しばしば適切です。
ただ、怒りを**情報**として使えるかどうかが、問いになることがあります——
この怒りは何を指しているか？
